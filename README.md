🌐 Portfolio Personal - Antonella Dev

¡Bienvenido/a al repositorio de mi portafolio web personal! Este proyecto es un sitio web moderno, responsivo y completamente interactivo diseñado para mostrar mi trabajo como desarrolladora Frontend, mis proyectos más destacados, tecnologías que domino y mi trayectoria actual.

---

✨ Características

* **Diseño Moderno y Futurista:** Interfaz fluida con una estética *dark mode* y acentos en tonos morados/neón.
* **Sección "Sobre mí" Interactiva:** Incluye una simulación de terminal con comandos (`whoami`, `stack`, `ubicacion`).
* **Galería de Proyectos:** Espacio dedicado para destacar desarrollos clave como *Personal Blog*, *Radio Cloud*, *Unika Web*, *Le Petit Bon Appétit*, entre otros.
* **Formulario de Contacto:** Integración limpia y accesible para conectar fácilmente vía email o redes sociales.
* **Completamente Responsivo:** Adaptado para visualizarse correctamente en dispositivos móviles, tablets y computadoras de escritorio.

---

🛠️ Tecnologías Utilizadas

El proyecto fue construido utilizando tecnologías web estándar, logrando un alto rendimiento sin dependencias pesadas:

* **HTML5** - Estructura semántica del sitio.
* **CSS3** - Estilos personalizados, diseño adaptativo (Flexbox/Grid) y animaciones.
* **JavaScript (Vanilla)** - Interactividad, lógica de navegación y componentes dinámicos.
